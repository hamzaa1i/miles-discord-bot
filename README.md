# 🤖 Miles - Discord Bot

Your personal AI companion and productivity assistant!

## Features

- 💰 **Economy System** - Work, earn, and spend coins
- 🏆 **Leaderboards** - Compete with other users
- 🧠 **AI Chat** - Talk to an intelligent AI
- 📝 **Productivity** - Notes, reminders, mood tracking
- 🎮 **Fun Games** - Dice, coin flip, RPS, memes
- 🎵 **Music Status** - Show Spotify activity
- 📊 **Rich Embeds** - Beautiful UI for all commands

## Quick Start

1. Clone this repository
2. Install dependencies: `pip install -r requirements.txt`
3. Copy `.env.example` to `.env` and add your bot token
4. Run: `python main.py`

## Commands

Use `/help` in Discord to see all available commands!
